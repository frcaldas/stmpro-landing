Perguntas frequentes

Como funciona a geração automática de mensalidades?
Todo dia 1º o sistema percorre todos os atletas ativos e gera uma cobrança para cada um. Você não precisa fazer nada — nem clicar num botão. As mensalidades já saem com o vencimento, valor e método de pagamento de cada atleta.
O atleta pode pagar por Pix, boleto e cartão?
Sim. Você configura a preferência de cada atleta no cadastro. Cada lembrete de WhatsApp já chega com o link correto para o método dele — sem confusão e sem o atleta precisar pedir.
Como funciona a confirmação automática de pagamento?
Quando o atleta paga, o sistema identifica o pagamento em tempo real e atualiza automaticamente. O STM Pro marca a cobrança como paga, cancela todos os lembretes pendentes daquele atleta e dispara uma mensagem de confirmação no WhatsApp dele — tudo sem você fazer nada.
O sistema funciona para personal trainers autônomos também?
Sim! O STM Pro foi construído tanto para CTs com múltiplos professores quanto para personals autônomos que atendem alunos avulsos e mensalistas. Cada conta tem seu próprio painel isolado, financeiro separado e configurações independentes.
Consigo importar meus alunos de uma planilha?
Sim. O sistema tem importação em massa via CSV — você exporta sua planilha atual, formata em dois minutos e importa tudo de uma vez. Nossa equipe auxilia no processo sem custo adicional.
Como funciona o Gympass, Totalpass e Wellhub?
Para atletas com benefício corporativo, o sistema registra o convênio no cadastro e o repasse é calculado separado do financeiro padrão. A integração com a API do Wellhub para repasses automáticos está disponível para CTs credenciados.
